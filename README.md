# Gandi-ddns

This is a simple python program that updated A and AAAA recornds using GANDI API.

## Requirements
python3.10 or more

## Installation

Download the repository

```bash
git clone https://gitlab.com/benmi/gandi-ddns.git
```
create a .venv
```bash
cd gand-ddns
python3 -m venv .venv
```
